## Features
* Project Builder
* Project Manager
* Project Settings
* Code Completion 
* Code Definition 
* Code Formatter
* Code Navigation
* Code Refactoring
* Code Suggestion 
* Snipptes
* Syntax Coloring
* Diagnostics
